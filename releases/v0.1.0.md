# helm-hooks v0.1.0

## Release Summary

Initial release of helm-hooks - a Helm post-renderer that enhances hooks with per-hook weights and environment injection.

### Features

- **Per-hook weights**: Assign different weights to different hook events from a single resource
- **Multi-hook splitting**: Automatically splits resources with multiple hooks into separate resources
- **Environment injection**: Injects `HELM_HOOK_EVENT` and `HELM_HOOK_WEIGHT` into containers
- **Auto-generate hooks**: Can derive hooks from `helm.sh/hook-weights` annotation
- **Kubernetes-safe naming**: Enforces 63-character limit with deterministic hashing
- **Validation**: Fail-fast on invalid hooks, duplicate hooks, or weight mismatches

### Supported Annotations

| Annotation | Description |
|------------|-------------|
| `helm.sh/hook` | Hook events (comma-separated) |
| `helm.sh/hook-weight` | Weight(s) for hooks |
| `helm.sh/hook-weights` | Explicit per-hook weights (e.g., `pre-install=-10,post-install=100`) |
| `helm.sh/hook-env` | Enable/disable env injection (default: `true`) |
