## Installation

### Helm 4 (Plugin)

```bash
helm plugin install https://github.com/AK121120/helm-hooks-plugin
```

### Helm 3 (Binary)

```bash
# Linux (amd64)
curl -sSL https://github.com/AK121120/helm-hooks/releases/download/v1.0.0/helm-hooks-linux-amd64 -o helm-hooks
chmod +x helm-hooks

# macOS (arm64)
curl -sSL https://github.com/AK121120/helm-hooks/releases/download/v1.0.0/helm-hooks-darwin-arm64 -o helm-hooks
chmod +x helm-hooks
```
